# Reverse Linked List

# Question

Given the head of a singly linked list, reverse the list, and return the reversed list.

# Example 1


Input: head = [1,2,3,4,5]
Output: [5,4,3,2,1]

# Example 2


Input: head = [1,2]
Output: [2,1]

# Example 3

Input: head = []
Output: []

# Introduction & Problem Understanding

Simple question, simple solution.
The linked list just needs to link the other way around. To do this, there are three parts that need to know

  1. A(previous node, it will be null at the start.)
  2. B(current node)
  3. C(next node)

From
null -> A -> B -> C -> D -> ...

To
null <- A -> B -> C -> D -> ...

Then
null <- A <- B -> C -> D -> ...

continue until the head === null
